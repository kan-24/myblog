# AGENTS.md

## 📖 Overview
这是一个使用 **Vue 3** + **Naive UI** 构建的博客网站，支持文章发布、分类管理、评论系统和响应式布局。  
目标是提供简洁优雅的界面和流畅的用户体验。

## 🛠 Setup commands
- 安装依赖: `pnpm install`
- 启动开发服务器: `pnpm dev`
- 运行测试: `pnpm test`
- 构建生产版本: `pnpm build`

## 🎨 Tech stack
- **Vue 3** + `<script setup>` 语法
- **Naive UI** 组件库
- **TypeScript** 严格模式
- **Vite** 构建工具

## 📏 Code style
- 使用单引号，末尾不加分号
- 组件命名采用 PascalCase
- 样式使用 CSS Modules 或 Scoped CSS
- 尽量使用组合式 API 和函数式模式

## 🔄 PR rules
- 所有代码必须通过 ESLint 检查
- 所有单元测试必须通过
- 提交信息遵循 [Conventional Commits](https://www.conventionalcommits.org/)
